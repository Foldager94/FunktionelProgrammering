## How to handle polymorphism

> The way we handle polymorphism is with the Maybe tag and Nothing
> In elm and haskell we can use the 'Maybe' tag.
>
> ```
> type Maybe x
>   = just x
>   | Nothing
> ```
>
> ```
> data Maybe x
>   = Nothing
>   | just x
> ```

## How to handle errors
